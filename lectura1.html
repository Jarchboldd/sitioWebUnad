¿Qué es HTML?

HyperText Markup Language.

Es el lenguaje base para casi todo el contenido Web. La mayor parte de los contenidos de Internet que ves en la pantalla del navegador están descritas, fundamentalmente, usando HTML. Más exactamente, podemos decir que el HTML es el lenguaje que describe la estructura y el contenido semántico de un documento o página Web.

Cuando miras una página web en el navegador, lo que ves a simple vista son palabras. Estas palabras normalmente tienen algunas características, como diferentes tamaños y colores. En algunos casos la página web puede mostrar imágenes o incluso vídeos. Otras veces, formularios donde puedes introducir o buscar información, o personalizar la apariencia de la página que estás viendo. La página también puede contener animaciones y contenidos que cambian mientras el resto de la página se mantiene sin cambios.

El lenguaje HTML puede ser creado y editado con cualquier editor de textos básico, como puede ser Gedit en GNU/Linux, el Bloc de notas de Windows, o cualquier otro editor que admita texto sin formato como GNU Emacs, Microsoft Wordpad, TextPad, Vim, Notepad++, entre otros.
