Programación web y la vinculación con base de datos.

Universidad Nacional Abierta y a Distancia.

Objetivo general.

Introducir en el entorno de la programación Web y comprender conceptos fundamentales mediante el uso de material audiovisual.

Objetivo específico 1.

Comprender la importancia de la vinculación de una página web con una base de datos.

Objetivo específico 2.

Mostrar la programación y los elementos necesarios para la vinculación de una página web con PHP a una base de datos.

Objetivo específico 3.

Efectuar acciones de consulta, edición, modificación y eliminación de datos y/o registros en una base de datos por medio de una página web con PHP.
